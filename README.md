# Test_task
## Установка
1. Клонируйте репозиторий:

   ```bash
   git clone https://github.com/Eldos7772002/Test_Task_Django_news_website.git
Перейдите в папку проекта:

    cd Test_task
Перейдите в папку проекта:

    virtualenv venv
    source venv/bin/activate
Примените миграции:

    python manage.py migrate
Запустите сервер:

    python manage.py runserver

Логин и пароль для админа:

      Admin
      Admin

Эта команда установит все зависимости, перечисленные в файле requirements.txt

      pip install -r requirements.txt


Для запуска предложения можно использовать докер 

      docker-compose up
